Types

Types provide a means of specifying constraints on a program and to some extent, the intent of that program.

There are two forms of types provided by nth: "inductive" and "qualified".

A inductive type is a type with a cardinality such that every possible instance is a member.

A qualified type is a type with an uncountable cardinality such that for every member, an instance can be created which is not a member.

Constraints

A constraint implies a process which is external to the semantics given for evaluation.

A program will either satisfy or violate a constraint.

In the case where a program violates a constraint, it is considered an error and that program must not be evaluated.

A conforming implementation must prove that all constraints are satisfied before evaluating a program. The approach used to prove a program satisfies all constraints is implementation dependent.

A constraint must only apply to the scope in which the constraint was asserted.

Constraints must be satisfied for the duration of the scope in which they were asserted.

An assertion may be made on a symbol, constraining it as an identifier, before that symbol is used as an identifier in a binding. In this case, assertion on that symbol implies that it is an identifier in a binding where the exact value is not known.

Assertion will apply a constraint to a program in two ways, by constraining the result of an evalaution or by constraining a binding.

Assertions which constrain a result enforce that the result must satisfy the constraint before the expression which produces the result is evaluated.

Assertions which constrain a binding enforce that each occurrance of the identifier in the scope where the constraint was asserted must satisfy the constraint.

Correct Programs

A program is determined to be correct if there are no errors resulting from constraint violations.

A constraint is either conformed to or violated by a program according to a set of rules which specify how a constraint relates to a program.

Type Rules

For a program to be determined correct, it must conform to some rules of judgement which are defined such that a correct program cannot cause a contradictory judgement. These rules are given below in a grammar which corresponds to the default expressions provided by nth.
