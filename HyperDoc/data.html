Data

Data is the substance which operations are executed on.

A unit of data which has been instantiated will always consume resources in the form of a size measured in Bytes.

The resources consumed by an instantiated unit will be constant for the duration of that unit.

Discarding a unit must releast all resources which were consumed by instantiation.

Scalars

Bytes

The Byte is the smallest unit of data, such that one Byte corresponds to one resource in the environment.

A Byte may store one of a range of whole, positive numberical values.

The minimum possible value which can be stored in a Byte is zero.

The maximum possible value which can be stored in a Byte is implementation dependent but should correspond to the largest value which can be stored in the smallest unit of storage for the computer targetted by the implementation.

Places are derived from Bytes such that for each Byte provided by the computer, there is a unique Place

Recursion

Recursion may occur in data where a component of some data may refer to that data. In this case, recursion is only observed by the operations which access components of that data. There will only ever be a single data unit. Executing operations which access data from a component which contains a recursive reference will only yield the existing data unit which contains the recursive component.
