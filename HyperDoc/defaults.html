Expressions

Expressions which are provided by default are implemented as transforms, but in an implementation defined manner.

Default expressions must be correct according to the transform types provided for each default expression.

The expressions given below are provided by the default image as if they were bound in an image using a definition expression.

The function representing the transform which rewrites a default expression is implementation dependent.

The type Scalar must be the supertype of any form of data corresponding to some implementation dependent format for storing information in the environment. For example, the type Word corresponds to a format which must be provided by the environment, but the type function corresponds to an abstract form of information specific to nth.
