Overview

Scope

1-1

This document concerns a programming language, nth, and its meaning, in order to support the portability of nth programs across a variety of systems.

1-2

This document specifies:

  • The form of an nth program.
  • The format of an nth program.
  • The effects of interpreting an nth program.
  • The process by which an nth program is judged correct.
  • The notion of a compliant implementation.
  • The resources which are provided by an implementation.
1-3

This document does not specify:

  • The method of interpretation of an nth program.
  • The means by which an nth program is invoked.
  • The effect of unspecified meanings.
  • The significance or correspondence of a correct program to its execution environment.
1-4

For the purposes of this standard, the nth programs which are provided assume an implementation executed on a computing system posessing infinite speed and resources. Equivalent programs with alternative resource requirements may be determined, at least to correctness, by a conforming implementation.

Conventions

Format

3-1-1

The nth Programming Language HyperDoc is formatted using a hypertext medium. Each passage is given a unique hyperlink and may contain hyperlinks to other passages and content. Sections are presented as separate pages.

Navigation

3-2-1

The three buttons at the bottom of each page allow navigation between each page. The bottom-left button (☚) will navigate to the previous section. The bottom-right button (☛) will navigate to the next section. The bottom-center button (·) will navigate to the cover page.

3-1-3

The cover page includes a search navigation feature. Section names, terms, and passages which have links are all indexed and searchable.

When a term is entered, a list of search results is given below the search entry. Items in this list are links and can be clicked. Additionally, pressing the enter key, or clicking the "Go" button will automatically navigate to the first result.
