1-4

Evaluation of a program is performed by locating expressions which contain only data. In other words, they are are non-nested Program Model components. Once an expression of this form is found, it is evaluated and replaced in the Program Model by its data result. This process is repeated until the Program Model is comprised of only a single data result.

Contexts

2-1

A context contains the bindings which have been established by the default function Let or by function application.

The process which resolves a binding is executed by applying the context to the symbol or expression and replacing that symbol or expression with the result of the application.
