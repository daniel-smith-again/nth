Evaluation

Evaluation determines the result of a program through a process of repeated intermediary steps which approach a final data result.

Given an expression, there are two steps which are performed to determine the result, Rewriting and Reduction.

Rewriting

An expression may either correspond to a default expression, or an expression which can be transformed.

If an expression is not a default expression, then the rewrite step will be performed.

If a rewrite step is performed, it will check the image for a transform which can be applied to the expression.

If no transforms can be applied to the expression, it is considered an error.

If an expression with one or more nested expressions is rewritten, each inner expression must be rewritten before the outer expression.

Reduction

A program can be reduced only when all expressions in the program are default expressions.

Reduction begins by attempting to execute the top, outermost expression in the program.

Contexts

The Context is used to persist data for one or more operations.

Contexts are provided by images, application, and the default function, do.
