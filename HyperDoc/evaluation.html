Evaluation

1-1

Evaluation occurs in three discrete phases: "Rewrite", "Resolve", and "Reduce".

1-2

Evaluation is performed by determining how a program may be replaced with data resulting from execution.

1-3

The result of evaluating a program must always result in data.

The Rewrite Phase

2-1

The Rewrite Phase resolves any higher order bindings by applying the higher order binding function to the expression and substituting it with the result.

The Resolve Phase

3-1

The Resolve Phase resolves any bindings by replacing the symbol in the Program Model with the data associated with that binding.

The Reduce Phase

4-1

The Reduce Phase determines the data resulting from executing the Program Model.
