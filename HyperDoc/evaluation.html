Evaluation

Evaluation determines the result of a program through repeated reductive steps where Program Model components that imply operations are executed and replaced with their data result.

Evaluation is performed according to discrete steps which transform the structure of a Program Model. For any given program which is being evaluated, the steps are...

  1. Locate the first expression within the program which contains only data. In other words, the first expression which does not contain another expression as a field.
  2. Repeatedly query the context for bindings of that expression, replacing the expression with the bound data for each successful query until there are no successful queries.
  3. For each symbol in the expression, query the context for bindings of that symbol and replace it with the bound data.
  4. Execute the operation associated with that expression and replace the expression with the result in the field that contains it.

Contexts

A context is formed according to the extent of an image or function. Each function and image will form a unique context containing the bindings established by their construction.
