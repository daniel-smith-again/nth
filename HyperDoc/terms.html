nth

A programming language; also written as nth.

Syntax

The correspondence between a source and a correspondenceProgram Model.

Computer

The device or mechanism which is responsible for executing a program.

Glyph

A character under one of the General Category Major Classes L, M, P, S, or C in 4.5 General Category, Table 4-4 of the Unicode® Standard

Digit

A character under the General Category Major Class N in 4.5 General Category, Table 4-4 of the Unicode® Standard

Blank

A character under the General Category Major Class Z in 4.5 General Category, Table 4-4 of the Unicode® Standard

Program Model

The data structure which forms the cannonical representation of an nth program.

Data

A static information constant of some size occupying storage resources.

Storage

The environment resource which is consumed when generating or holding information during execution of a program.

Resource

A facility of the environment that is consumed by the process of executing a program.

Program

The instructions which execution uses to carry out effects in the environment.

Environment

That which is provided by the computer to facilitate execution.

Execute

The process of carrying out effects in the environment according to a program.

Symbol

An abstract, unique data value represented by a textual entity.
