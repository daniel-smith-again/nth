Terms

Application

The operation which invokes a function (Refer to the section on Functions).

Blank

A character under the General Category Major Class Z in 4.5 General Category, Table 4-4 of the Unicode® Standard

Character

An encoded numerical value according to the Unicode® Standard or the ASCII Standard corresponding to written text.

Collection

The construct provided by nth for creating structured data (Refer to the section on Collections).

Computer

The device or mechanism which is responsible for executing a program.

Data

A static information constant of some size occupying storage resources.

Default

Data or another resource which must be present as auxillary to the implementation.

Digit

A character under the General Category Major Class N in 4.5 General Category, Table 4-4 of the Unicode® Standard

Element

A single component of a collection.

Environment

That which is provided by the computer to facilitate execution.

Execute

The process of carrying out effects in the environment according to a program.

Expression

A syntax which denotes an operational step.

Function

The construct provided by nth for creating re-useable programs (Refer to the section on Functions).

Glyph

A character under one of the General Category Major Classes L, M, P, S, or C in 4.5 General Category, Table 4-4 of the Unicode® Standard

Implementation

A program which acts according to the requirements given in this document (refer to the section on Conformance).

Member

A single component of a type.

Operation

A single executional step, denoted by an expression.

Program

The instructions which execution uses to carry out effects in the environment.

Program Model

The data structure which forms the cannonical representation of an nth program.

Resource

A facility of the environment that is consumed by the process of executing a program.

Source

A program format representing a Program Model as a string.

Storage

The environment resource which is consumed when generating or holding information during execution of a program.

StringA collection of character values.
Symbol

An abstract, unique data value represented by a textual entity.

Syntax

The correspondence between a source and a Program Model (Refer to the section on Syntax).

Type

The construct provided by nth for reasoning and constraining the possible instantiations of data in a program (Refer to the section on Types
