Foreword

1-1

This document forms a comprehensive specification of the properties of an abstract system for describing and computing programs, "nth" (stylized as nth).

1-2

The purpose of this document is to encourage the portability if nth programs across a variety of computing systems and implementations.

Conformance

2-1

This document represents a standard of conformance. As such, the descriptions therein represent requirements for an implementation of nth.

2-2

An implementation of nth is said to conform to this standard if it comprehensively demonstrates all the properties described in this document.

2-3

In cases where the practical limitations of an otherwise conforming implementation prevent full conformance, the implementation must provide documentation about which clauses or sections it does not conform to.
