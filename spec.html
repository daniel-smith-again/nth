Overview

This document provides the definition the programming language, "nth", stylized as nth. Specifically, the meaning, interpretation, and formation of programs written in the nth programming language. Information is provided in such a way that the reader is able to rely on this document not only to understand how nth is used, but also how one might implement nth.

Terminology

Host
The arbiter of computing resources in the environment where an implementation executes programs.
Processor
The agent responsible for executing a program.
Execution
The process by which the meaning of a program is carried out.
Program
The data to be interpreted during execution.
Context
The enumeration of all bindings.
Binding
An association between a symbol and some data such that the data may replace the symbol in a program anywhere that association is present.
Symbol
An opaque data type which is a component of a program.
Expression
A composite data type which is a component of a program.
Function
A program which preserves a context to be used during execution.
Syntax
The sequence(s) of characters which correspond to a program.
Character
An opaque value corresponding to a visually rendered grapheme.
